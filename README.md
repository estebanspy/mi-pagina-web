# Mi Aplicación Web

Aplicación web desarrollada como parte de la Unidad 6 del módulo de Despliegue de Aplicaciones Web del ciclo formativo DAW.

## Propósito

Este proyecto tiene como objetivo demostrar el uso de Git y GitHub para el control de versiones, junto con Jenkins para la integración continua. La aplicación consiste en una página web estática con HTML y CSS que sirve como base para el pipeline de CI/CD.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git / GitHub
- Jenkins

## Estructura del proyecto

```
mi-pagina-web/
├── index.html   → Página principal
├── style.css    → Estilos de la aplicación
└── README.md    → Documentación del proyecto
```
